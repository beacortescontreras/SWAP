#1.- Buscar información sobre el tipo de contenido para el que los siguientes software de servidor web es más adecuado (o más eficiente, o se usa más actualmente…):
#- Apache
#- nginx
#-thttpd
#-Cherokee
#-node.js

Tanto Apache como Nginx son dos servidores web de los más potentes y eficaces. El servidor más usado actualmente es Apache seguido de Nginx. Apache tiene bastante poder y
Nginx velocidad, aunque también tienen inconvenientes por ejemplo Apache consume demasiada memoria del servidor y Nginx necesita ayuda para servir contenido dinámico.
Thttpd, es un servidor bastante rápido, ligero y seguro. A comparación con Apache, en situaciones de carga extrema su rendimiento es más alto. Su uso principal suele ser el de servidor rápido de contenido estático.
Por otra parte tenemos Cherokee, el cual es destacado por ser un servidor ligero, rápido y von una interfaz de administración muy cuidada, sirve contenido estático. Implementa una librería para dotar a toda clase de aplicaciones de servicios web de forma fácil y rápida. En su desarrollo se hace un especial esfuerzo en mantener un core reducido para poder ser utilizado en sistemas empotrados.
Finalmente tenemos node.js, es un sistema para programación de eventos que se puede usar como intérprete de JS.
